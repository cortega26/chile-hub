Un modelo de datos nacional, capa por capa

Cada capa nueva pasa un filtro de utilidad real, estabilidad de la fuente y licencia verificable. Lo que está publicado funciona; lo que está en evaluación, todavía no.

Uso rápido

Consume archivos publicados sin clonar el repositorio ni instalar el helper.

Python + Polars

Carga el Parquet publicado directamente desde cualquier notebook o servicio.
import polars as pl

url = "https://tooltician.com/chile-hub/data/normalized/comunas.parquet"
df = pl.read_parquet(url)

DuckDB directo

Consulta el archivo remoto con SQL, sin descargarlo manualmente.
SELECT *
FROM read_parquet('https://tooltician.com/chile-hub/data/normalized/comunas.parquet')
LIMIT 10;

JSON + curl

Descarga un artefacto abierto con una herramienta disponible en cualquier entorno.
curl -L "https://tooltician.com/chile-hub/data/normalized/comunas.json" \
  -o comunas.json
