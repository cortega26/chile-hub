Uso rápido
Tres formas simples de empezar a consumir el hub según tu flujo de trabajo.
Python + helper
Ideal si quieres descubrir capas y cargar Parquet sin recordar paths manualmente.

hub = ChileHub()
print(hub.list_datasets())
df = hub.load_polars("comunas")
